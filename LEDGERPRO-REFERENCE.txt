================================================================================
  1. WHAT IS LEDGER PRO
================================================================================

Ledger Pro is a cloud-based double-entry accounting platform built as a
Software-as-a-Service (SaaS). It runs entirely in the cloud — no installation
required. Users access it through a web browser at ledger.nexvarlab.com.

TYPE: Cloud Software / SaaS (Software as a Service)
STACK: Next.js 14 + PostgreSQL + Prisma + NextAuth.js + Recharts
PRICING: Free Trial → Basic $29/mo → Pro $79/mo → Enterprise $199/mo

  3. PLATFORM DETAILS
================================================================================

A. GITHUB — Code Repository & Version Control
   URL: https://github.com/mjawaidca-prog/ledgerpro
   Purpose: Stores all source code, tracks changes, enables collaboration
   How it works: Code is pushed from local machine to GitHub. Vercel watches
   the main branch and auto-deploys on every push.

B. VERCEL — Hosting & Deployment
   URL: https://vercel.com (ledgerpro project)
   Purpose: Hosts the Next.js application, serves web pages, runs API routes
   Domains: ledgerpro-theta.vercel.app (auto) + ledger.nexvarlab.com (custom)
   Build Command: npx prisma generate && next build
   Deploys: Automatic on every git push to main branch
   Environment Variables: DATABASE_URL, DIRECT_URL, NEXTAUTH_SECRET,
                          NEXTAUTH_URL, STRIPE_SECRET_KEY, STRIPE_WEBHOOK_SECRET,
                          RESEND_API_KEY

C. SUPABASE — Cloud Database
   URL: https://xdniuuetrsflynxvnqzc.supabase.co
   Purpose: PostgreSQL database storing all accounting data
   Contains: Users, companies, invoices, bills, transactions, journal entries,
             chart of accounts, subscriptions, tax rates, audit logs
   Connection: Direct PostgreSQL connection (port 5432)

D. CLOUDFLARE — DNS & Security
   URL: https://dash.cloudflare.com (nexvarlab.com zone)
   Purpose: DNS management, CDN, SSL/TLS, DDoS protection
   Records: CNAME ledger → cname.vercel-dns.com (proxied, orange cloud)
            Same setup as pay.nexvarlab.com

E. NAMECHEAP — Domain Registrar
   Purpose: Owns the nexvarlab.com domain registration
   DNS: Delegated to Cloudflare (nameservers point to Cloudflare)
   No changes needed at Namecheap for new subdomains

F. STRIPE — Payments & Subscriptions
   URL: https://dashboard.stripe.com
   Purpose: Handles recurring subscription billing, payment processing
   Plans: Free Trial ($0), Basic ($29/mo), Pro ($79/mo), Enterprise ($199/mo)
   Webhook: POST /api/stripe/webhook handles subscription lifecycle events

G. RESEND — Transactional Email
   URL: https://resend.com
   Purpose: Sends invoice emails, payment confirmations, notifications
   Status: Not yet configured (placeholder key in env)

  5. KEY FEATURES
================================================================================

ACCOUNTING ENGINE
  • Double-entry journal (every transaction has debit + credit)
  • Chart of Accounts with 5 GL types (Asset, Liability, Equity, Income, Expense)
  • Trial Balance verification (debits must equal credits)
  • Period Close with 409 Conflict guard on closed periods

CANADIAN TAX SYSTEM
  • GST/HST/PST for all 13 provinces/territories
  • Auto tax calculation based on company province
  • Business Number (BN) and GST/HST Number on company profile
  • Tax rates: AB 5% | BC/MB 12% | SK 11% | ON 13% | NB/NL/NS/PE 15% | QC 14.975%

MULTI-TENANT SAAS
  • Users can belong to multiple companies (accountant use case)
  • Company switcher with session isolation
  • Role-based access: Owner, Admin, Bookkeeper, Viewer
  • Audit log tracks every mutation with user + timestamp

BANKING
  • Financial accounts (checking, savings, credit card)
  • CSV/OFX import with column mapping profiles
  • Bank reconciliation (two-column view, batch reconcile)
  • Transfer matching between accounts

AI CATEGORIZATION
  • Pattern-based rules (merchant match, description, amount range, regex)
  • Auto-apply rules against unreviewed transactions
  • Priority ordering, match count tracking

RECURRING TRANSACTIONS
  • Weekly/monthly/quarterly/annual templates
  • Auto-post with next date calculation
  • Journal line builder with GL account picker
